Repositorios Descripccion
Git_Hub Github Es un portal creado para alojar el código de las aplicaciones de cualquier desarrollador, y que fue comprada por Microsoft en junio del 2018. La plataforma está creada para que los desarrolladores suban el código de sus aplicaciones y herramientas, y que como usuario no sólo puedas descargarte la aplicación, sino también entrar a su perfil para leer sobre ella o colaborar con su desarrollo.
TaraVault TaraVault es una plataforma de gestión de código fuente desarrollada por Inflectra, que ofrece repositorios seguros y accesibles para Git y Subversion. Está integrada con las soluciones de gestión del ciclo de vida de aplicaciones (ALM) de Inflectra, como SpiraPlan y SpiraTeam, lo que permite una trazabilidad completa entre el código fuente y otros artefactos del proyecto. INFLECTRA Entre las características destacadas de TaraVault se incluyen: Alojamiento de código fuente de nivel empresarial: Cada proyecto en TaraVault cuenta con su propio repositorio privado, ya sea en Git o Subversion, adaptándose a las necesidades específicas de cada equipo. INFLECTRA Integración con Spira: La plataforma se integra perfectamente con las herramientas ALM de Inflectra, permitiendo la vinculación de requisitos, tareas, defectos y problemas con archivos y revisiones de código fuente, lo que facilita una gestión integral del desarrollo de software. INFLECTRA Navegación y revisión de código: Ofrece herramientas para explorar el código fuente, visualizar diferencias en el código (diffs) de manera en línea o lado a lado, y gestionar solicitudes de extracción (pull requests), mejorando la colaboración entre los desarrolladores.
